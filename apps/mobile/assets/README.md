# Assets

Placez vos assets ici :

- `icon.png` - Icône de l'application (1024x1024)
- `splash.png` - Écran de démarrage (1242x2436)
- `adaptive-icon.png` - Icône adaptative Android (1024x1024)
- `favicon.png` - Favicon pour le web (48x48)

## Générer les assets

Vous pouvez utiliser des outils en ligne pour générer vos assets :
- [App Icon Generator](https://www.appicon.co/)
- [Expo Asset Generator](https://github.com/expo/expo-cli)

Ou créez-les manuellement avec vos outils de design préférés.
